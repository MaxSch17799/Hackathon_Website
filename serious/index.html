Event snapshot

  • Purpose: experience coding with AI at work and share tips/tricks.
  • Audience: everyone from beginner to experienced coder.
  • Team size: 3-4 people.
  • Theme: examples are provided for inspiration, but anything goes if it teaches AI tooling usage.
  • Kiosk digital implementation; make it easier to keep track of kiosk sales and automate the invoicing.
  • Warehouse QR; create a system that allows for easier real life tracking of where components are.
  • Website Rainforest; modernize and update the public website.
  • Dashboard, but great; improve the dashboard.
  • Dishwasher state machine; Digitalize the dishwasher schedule and make the state machine digital.
  • Create VS code extension for...
  • Demo time limit: 5 minutes per team.
  • Judging tease: best demo, best use of AI, most useful, funniest, etc. + a mystery judge.
  • Prize: a trophy.
  • Dietary restrictions: Contact Allard.
